# 中药材智能识别系统

这是一个基于深度学习和人工智能技术的中药材识别系统，整合了本地深度学习模型和百度AI云端识别能力，使用Flask作为后端框架，TensorFlow作为深度学习引擎，提供准确的中药材识别服务。系统通过双重识别引擎相互验证，显著提高了中药材识别的准确性和可靠性。

![系统截图](static/images/system_screenshot.png)

## 系统介绍

本系统旨在帮助中医药相关人员和爱好者快速识别中药材，获取药材详细信息，辅助学习和工作。系统结合了本地训练的深度学习模型和百度AI接口，提供双重识别结果，大幅提高识别准确率。

### 系统工作原理

1. **图像采集与预处理**：
   - 系统接收用户上传或拍摄的药材图像
   - 对图像进行标准化处理（调整大小、颜色校正等）
   - 图像增强，提高特征可辨识度

2. **双重识别引擎**：
   - 本地深度学习模型（EfficientNetV2B0架构）进行初步识别
   - 百度AI通用物体识别API提供辅助识别
   - 结果对比与融合，提高识别准确性

3. **相似药材推荐**：
   - 基于药材特性（性味、功效等）进行相似性计算
   - 采用多维特征匹配算法，找出相似药材
   - 按相似度排序，提供参考建议

4. **数据管理与学习**：
   - 识别历史记录存储，方便用户复习与查询
   - 识别反馈机制，不断优化模型性能
   - 识别数据分析，提供药材使用趋势报告

## 安装指南

### 环境要求
- 操作系统：Windows 10/11, macOS 10.15+, 或 Ubuntu 18.04+
- Python 3.8 或更高版本
- pip 包管理器 (20.0+)
- 至少2GB可用存储空间
- 至少4GB RAM内存
- 互联网连接 (用于百度AI接口调用)
- 推荐：NVIDIA GPU (用于模型训练和加速推理)
  - CUDA 11.0+ 和 cuDNN 8.0+

### 详细安装步骤

1. **克隆项目**
```bash
git clone https://github.com/yourusername/chinese_herb_recognition.git
cd chinese_herb_recognition
```

2. **创建虚拟环境(推荐)**
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python -m venv venv
source venv/bin/activate
```

3. **安装依赖**
```bash
# 更新pip
pip install --upgrade pip

# 安装基本依赖
pip install -r requirements.txt

# 如果有GPU，安装GPU版本的TensorFlow (可选)
pip install tensorflow-gpu
```

4. **配置百度AI接口**
   - 注册[百度AI开放平台](https://ai.baidu.com/)账号
   - 创建图像识别应用，获取API Key和Secret Key
   - 在项目根目录创建`.env`文件(或修改config.py)，添加以下内容：
   ```
   BAIDU_API_KEY=your_api_key_here
   BAIDU_SECRET_KEY=your_secret_key_here
   ```

5. **创建必要的目录**
```bash
# 创建上传文件存储目录
mkdir -p static/uploads/detections
mkdir -p static/uploads/herbs

# 创建模型存储目录
mkdir -p models
```

6. **下载预训练模型**
   - 从[此链接](https://github.com/yourusername/chinese_herb_recognition/releases)下载预训练模型文件
   - 将下载的`herb_model.h5`文件放入`models`目录

7. **初始化数据库**
```bash
# 初始化迁移仓库
flask db init

# 创建迁移脚本
flask db migrate -m "Initial migration"

# 应用迁移
flask db upgrade
```

8. **创建管理员账户**
```bash
# 运行创建管理员的脚本
python create_admin.py
```

9. **运行项目**
```bash
# 开发环境运行
python app.py

# 或使用Flask命令(设置开发模式)
export FLASK_APP=app.py
export FLASK_ENV=development
flask run
```

10. **访问系统**
   - 打开浏览器，访问 http://127.0.0.1:5000
   - 使用步骤8创建的管理员账户登录系统

### 生产环境部署(可选)

对于生产环境，建议使用以下配置：

1. **使用Gunicorn作为WSGI服务器**

pip install gunicorn

## 常见问题解答

1. **识别相关问题**

   **为什么识别结果不准确？**
   - 确保图片清晰、光线充足，避免反光或阴影
   - 尝试不同角度拍摄药材，展示其特征部位
   - 确保药材是完整的，而非碎片
   - 检查是否有网络连接问题(百度AI需要网络连接)
   - 同一药材可能有多种外观，尝试使用典型样本

   **如何提高识别准确率？**
   - 使用纯色背景(白色最佳)拍摄药材，减少干扰
   - 避免多种药材混合在一起
   - 适当调整光线，使特征清晰可见
   - 对于相似药材，聚焦拍摄特征部位
   - 定期更新系统，获取最新模型

   **为什么本地模型和百度AI结果不同？**
   - 两种模型训练数据不同，可能有各自的识别优势
   - 本地模型针对特定药材优化，百度AI通用性更强
   - 某些药材特征不明显，可能导致识别分歧
   - 综合考虑两种结果，参考置信度高的识别结果

2. **系统使用问题**

   **系统无法连接百度AI？**
   - 检查网络连接是否稳定
   - 验证API密钥是否正确设置
   - 确认百度AI服务是否可用(可能有调用限制)
   - 查看日志文件了解具体错误信息
   - 临时使用仅本地模型模式继续工作

   **上传图片失败？**
   - 确保图片格式支持(JPG、PNG、JPEG、GIF)
   - 检查图片大小是否超过限制(10MB)
   - 确认存储空间充足
   - 尝试刷新页面后重新上传
   - 如持续失败，尝试不同浏览器

   **系统运行缓慢？**
   - 检查服务器资源使用情况
   - 清理历史数据，优化数据库
   - 尝试减小上传图片尺寸
   - 确保模型加载正确
   - 考虑升级服务器配置

3. **账户相关问题**

   **如何重置密码？**
   - 在登录页面点击"忘记密码"
   - 输入注册时使用的邮箱
   - 按照收到的邮件指引重置密码
   - 如未收到邮件，检查垃圾邮件文件夹
   - 联系管理员协助重置

   **账户被锁定？**
   - 多次密码错误可能导致临时锁定
   - 等待30分钟后重试
   - 使用密码重置功能
   - 联系系统管理员解锁账户

4. **数据管理问题**

   **如何备份数据？**
   - 管理员可以在管理面板使用"数据备份"功能
   - 系统自动备份：每日凌晨3点自动执行
   - 手动备份：在命令行执行`python backup.py`
   - 备份文件存储在`backups`目录
   - 定期将备份文件转移到外部存储

   **丢失识别历史怎么办？**
   - 检查"已删除"文件夹，可能在那里找回
   - 使用最近的备份恢复数据
   - 联系管理员查询服务器日志
   - 如确实丢失，需要重新进行识别
