# 中药材小程序

## 项目简介
这是一个专注于中药材信息展示和查询的微信小程序。该小程序旨在为用户提供便捷的中药材信息查询服务，包括药材展示、搜索、详情查看等功能。

## 项目结构
```
├── app.js                 // 小程序入口文件
├── app.json              // 小程序全局配置
├── app.wxss             // 小程序全局样式
├── pages                // 页面文件夹
│   ├── index           // 首页
│   ├── list           // 列表页
│   ├── detail         // 详情页
│   └── login          // 登录页
└── utils               // 工具函数文件夹
    └── util.js        // 通用工具函数
```

## 功能模块

### 1. 首页模块 (pages/index)
- **搜索功能**
  - 支持中药材名称搜索
  - 搜索结果跳转到列表页展示
  
- **轮播图展示**
  - 展示精选中药材图片
  - 支持自动轮播
  - 点击可跳转至详情页
  
- **公告资讯**
  - 展示最新公告信息
  - 包含标题和发布日期
  - 支持点击查看详情

### 2. 列表页模块 (pages/list)
- 展示搜索结果
- 支持药材分类展示
- 提供筛选和排序功能

### 3. 详情页模块 (pages/detail)
- 展示药材详细信息
- 包含药材图片、名称、功效等信息
- 提供收藏功能

### 4. 登录模块 (pages/login)
- 用户登录功能
- 个人信息管理

## 技术特点
1. 采用微信小程序原生开发
2. 使用组件化开发方式
3. 统一的样式管理
4. 模块化的代码组织

## 数据管理
- 使用全局数据管理（app.js中的globalData）
- 页面级数据管理（各页面的data对象）

## 开发规范
1. 文件命名规范
   - 页面文件：小写字母，多个单词用连字符（-）连接
   - 工具类文件：小写字母，多个单词用下划线（_）连接

2. 代码规范
   - 使用ES6语法
   - 统一的代码缩进（2空格）
   - 必要的代码注释
